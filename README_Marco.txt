To embed videos in the collection block, add the html snippet in the text flag, there you can also link the title of the video description to a specific page. Remember to not leave any white line
or the hugo compiler will report an error

To compile the website locally, just enter in the repository folder and type
hugo server -D
then when changes are done, add, commit and push into the github repo
